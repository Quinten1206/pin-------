# 📌 DeskPins — 窗口置顶工具

一个**免安装**的 Windows 小工具，用图钉📌的方式把任意窗口固定在屏幕最上层。

> 双击运行，右键托盘操作，不需要任何命令行。

---

## 🚀 怎么用？（3 步上手）

### 第 1 步：双击运行

找到 `DeskPins.exe`，**双击打开**。

- 没有任何窗口弹出 —— 这是正常的。
- 看一眼屏幕右下角的**系统托盘**（那个小三角 `^` 里面），会多出一个**红色图钉图标**。

### 第 2 步：钉住一个窗口

1. **右键**点击托盘里的红色图钉图标
2. 点 **"开始钉选"**
3. 你的鼠标指针会变成**一个红色图钉** 🖱️
4. 用这个图钉鼠标**左键点击**你想置顶的窗口（比如记事本、浏览器）
5. 搞定！那个窗口现在会一直浮在最上面

> 💡 在钉选模式下，**右键点击**或按 **ESC 键**可以取消，不钉任何窗口。

### 第 3 步：取消置顶

**方法一：点击标题栏上的红色方块**

被置顶的窗口，标题栏右上角会出现一个**红色小方块**。点击它就能取消置顶。

**方法二：全部取消**

右键托盘图标 → 点 **"清除所有置顶"** → 所有窗口恢复原样。

## 📋 托盘菜单说明

| 菜单项 | 作用 |
|--------|------|
| **开始钉选** | 进入图钉模式，点击窗口置顶（双击托盘图标同样效果） |
| **清除所有置顶** | 一次性取消所有已置顶的窗口 |
| **关于** | 显示版本信息 |
| **退出** | 关闭程序，同时取消所有窗口置顶 |

## ❓ 常见问题

**Q: 双击后什么都没发生？**

A: 看看右下角系统托盘的小三角 `^` 里面有没有红色图钉图标。这个程序**没有主窗口**，一切操作都在托盘图标上。

**Q: 为什么有些窗口标题栏上没有红色图标？**

A: 个别系统级窗口（如任务管理器、系统设置）出于安全限制不支持。但**置顶功能不受影响**——窗口还是会浮在最上面，只是标题栏没有图标。

**Q: 关了程序，之前置顶的窗口会怎样？**

A: 退出程序时会自动把所有窗口恢复原样。如果程序意外崩溃，被注入的小图标也会在几秒钟内自行清理。

**Q: 怎么彻底删除？**

A: 直接删除 `DeskPins.exe` 文件即可。程序不会在系统里留下任何残留（临时文件会在退出时自动清理）。

**Q: 为什么文件这么大？**

A: 因为把整个 .NET 运行环境都打包进去了，确保收到文件的电脑不需要装任何额外的东西就能直接运行。

## 🛠️ 开发者信息（可选阅读）

这是一个练手项目，模仿经典工具 [DeskPins](https://deskpins.en.softonic.com/)。

**技术栈：** C# / .NET 8 / WinForms + C++ DLL（MinGW-w64）

**自己编译：**
```bash
# 前置：需要 .NET 8 SDK + MinGW-w64 (g++)
cd HookDLL && cmake -B build -G "MinGW Makefiles" && cmake --build build --config Release
cp build/DeskPinsHook.dll ../DeskPins/Resources/
dotnet publish DeskPins -c Release -p:PublishSingleFile=true -p:SelfContained=true -o publish
```
